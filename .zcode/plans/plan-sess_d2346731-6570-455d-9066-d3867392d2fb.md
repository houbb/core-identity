# P1 MVP 实施计划

## 技术基线
- **数据库**: SQLite 开发/测试，MySQL 迁移并行维护
- **密码**: BCrypt（spring-security-crypto），最少 8 位无复杂度
- **通知**: Console 适配器，预留 core-notification 接口
- **会话**: 空闲 2h / 绝对 24h
- **安全**: 手动 Filter 实现 CSRF，不引入 Spring Security
- **前端**: Vue 3 + Pinia，复用现有 Apple UI Token

## 六阶段实施

### P1.1 用户与组织模型
- 9 张新表（SQLite + MySQL Flyway 迁移）
- Domain + Repository（接口+JDBC实现）×9
- PasswordHasher 接口+BCrypt 实现
- 添加 spring-security-crypto 依赖

### P1.2 注册与邮箱验证
- AuthService、OTP Token 服务、ConsoleNotificationAdapter
- 注册页(/register)、邮箱待验证页(/check-email)、验证结果页(/verify-email)
- 幂等注册、事务保护、通知失败不回滚

### P1.3 登录与会话
- SessionService + LoginAttempt + 临时锁定
- CSRF Filter（复用 FilterRegistrationBean 模式）
- 登录页(/login)、账户首页(/account)、路由守卫

### P1.4 个人资料与密码
- MeController (GET/PATCH me、修改密码、会话管理)
- 忘记/重置密码（一次性 Token 模式）
- 个人资料页、安全页、会话管理页

### P1.5 平台管理员
- Identity Backend Internal API 扩展（admin-auth + user CRUD）
- Admin Backend BFF 编排（不访问 DB）
- Admin Web：登录/用户列表/详情/创建/禁用/启用/撤销会话
- Bootstrap 管理员初始化脚本

### P1.6 安全加固 + 全面测试
- 限流 Filter、敏感数据脱敏、错误统一映射
- SQLite/MySQL 迁移回归、端到端测试、越权测试
- 更新 CHANGELOG + README

## 预计文件变更
~75 新增 + ~20 修改（跨 4 个子项目 + contracts + scripts）
